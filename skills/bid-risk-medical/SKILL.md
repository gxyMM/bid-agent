# 🏥 医疗设备招标风险审查指南 (Medical Bid Risk Guidelines)

作为医疗器械领域的标书专家智能体，在审查招标文件 (RFP) 和投标文件 (Bid) 时，必须强制执行以下深度行业规则。智能体必须确保本技能包作为一个“自包含”的完整检查清单，涵盖通用底线与行业特有雷区。

## 核心审查逻辑 (Core Rules)

### 一、 通用底线审查 (Universal Baseline Rules) - 致命废标项
在深入医疗专业细节前，必须首先跨文档核实以下 4 条通用红线（一票否决）：
1. **法人电子签与盖章要求 (U-01)**：核查法定代表人或授权委托人是否在所有规定位置（投标函、法定代表人身份证明、授权委托书等）进行了签字或加盖个人印章。若是电子标，严查法人CA锁与企业CA锁是否混用，以及是否遗漏电子骑缝章。
2. **投标有效期天数 (U-02)**：提取投标文件承诺的有效期天数，强制要求 `承诺天数 >= 招标文件要求天数`（绝不能少于要求天数）。
3. **品牌与型号一致性 (U-03)**：同一设备绝不填报多个备选品牌。必须跨文档（《报价一览表》、《详细报价单》和《技术响应表》）核对同一型号的品牌是否做到三方绝对一致。
4. **整机质保期承诺 (U-04)**：提取质保期限与对象。若招标要求“整机质保”，投标承诺必须明确覆盖“整机”且 `承诺年限 >= 要求年限`，禁止含糊表述（如“满足原厂标准”）或仅承诺核心部件质保。

### 二、 医疗行业深度合规审查 (Medical-Specific Rules)
1. **资质合规性绝对匹配 (M-01)**
   *   **营业执照范围**：核对投标人是否具备对应类别的《医疗器械经营许可证》或《第二类医疗器械经营备案凭证》。
   *   **产品注册证**：强制校验所有带型号的设备，其《医疗器械注册证》及附件上的**规格型号**是否与投标报价表中的型号**100%一致（含大小写和破折号）**。
   *   **辐射安全许可**：如果清单中包含 X射线、CT、核磁、放疗等放射性设备，必须核实是否提供了《辐射安全许可证》。

2. **授权链条追踪 (M-02)**
   *   **进口设备专核**：进口设备必须提供制造商出具的**针对本项目的唯一授权书**。如果中间经过总代，必须提供清晰的“原厂 -> 中国区总代 -> 投标人”的完整层级授权链证明（严查授权层级是否断裂，公章是否为原章）。

3. **耗材绑定与阳光采购 (M-03)**
   *   **专机专用耗材**：排查设备是否捆绑专机专用试剂或耗材。如果有，必须在商务标中明确承诺该耗材已在当地“阳光采购/省药械集中采购平台”挂网，且承诺挂网价格为全国最低/平均水平。若隐瞒耗材绑定或价格偏高，极易被投诉废标。

4. **技术与偏离红线 (M-04)**
   *   **FDA/CE 认证陷阱**：国内公立医院招标原则上不得强制要求进口认证（FDA/CE），若 RFP 中作为加分项，投标时附带的认证文件必须与产品型号严格对应。
   *   **售后响应红线**：医疗设备关乎生命，招标文件常要求“2小时内响应，24小时到达现场，48小时内解决故障”。智能体必须跨文档扫描《售后服务承诺书》，确保时间承诺只少不多。

## 工作流提示 (Workflow Prompts)
当提取到包含“医院、医疗器械、注册证、卫健委”等关键词时，立即加载本指南的 U-01 至 U-04 及 M-01 至 M-04 规则进行逐条核实，并在最终报告中输出【医疗合规性排查结论】。
